To play the Reversi game:
	Intellij IDE and run
	.exe but you need to have jre environment
Then the game would start:
	Enter "1", or "0" to play first, or second
	Enter (0 - 7) for the row and the column
Follow the hint to input the row and column, try your best 
Some comments for AI vs AI
Program would terminate when there is a win, or draw.
